### 240202今日vue学习
一、v-model原理，其实就是value和@input事件的简写  
二、.sync修饰符实现 子组件 与 父组件数据 的 双向绑定  
三、利用 ref 和 $refs，在组件内获取 dom 元素 或 组件实例  
四、Vue 是异步更新DOM  
    this.$nextTick(函数体)：等 DOM更新后，才会触发执行此方法里的函数体  
    **注意：**$nextTick 内的函数体 一定是箭头函数，这样才能让函数内部的this指向Vue实例
